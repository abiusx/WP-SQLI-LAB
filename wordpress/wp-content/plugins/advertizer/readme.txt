Advertizer is a Wordpress Advertizing manager, that allows you to create as many ads blocks as you need. Any image format, basic trase of clicks.
== Description ==

You can create any banner or use image from other server. You can create as many Ads blocks as you need. Easy to use - just create blocks and then use them in theme widgets. 

== Installation ==

1. Upload `advertizer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add Ads block with parameters you need.
4. Add widgets with needed blocks to your WP theme
